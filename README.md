# 🔥 Smart Fire Management System (STM32)

An embedded systems project based on **STM32 microcontroller** designed to detect fire hazards in real-time and trigger automated safety responses such as alarms, notifications, and actuator control.

---

## 📌 Project Overview

The Smart Fire Management System is an embedded safety system that continuously monitors environmental conditions using sensors to detect fire or smoke. When abnormal conditions are detected, the system responds immediately by activating alarms and safety mechanisms.

This project was developed as part of an **Introduction to Embedded Systems** course.

---

## 🎯 Objectives

- Detect fire and smoke conditions in real time
- Reduce response time in fire emergencies
- Trigger audible/visual alarms automatically
- Provide reliable and low-power embedded monitoring
- Demonstrate STM32-based sensor integration and control logic

---

## 🧠 System Features

- Real-time sensor monitoring (fire/smoke/temperature)
- Threshold-based decision system
- Alarm activation (buzzer / LED indicators)
- Embedded firmware running on STM32
- Low-latency interrupt or polling-based architecture
- Modular firmware structure (Drivers + Application layer)

---

## 🧰 Hardware Components

- STM32 Microcontroller (e.g., STM32F103 / STM32F4 series)
- Flame Sensor / IR Fire Sensor
- Smoke Sensor (e.g., MQ-2 / MQ-135)
- Temperature Sensor (optional depending on design)
- Buzzer (alarm output)
- LEDs (status indicators)
- Power supply module
- Breadboard & wiring setup

---

## 💻 Software & Tools
